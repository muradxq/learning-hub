# Cmake

- Difference Between CMakeLists.txt and .cmake Files
  - In CMake, both CMakeLists.txt and .cmake files define build configurations and scripts, but they serve different purposes.


Project Structure
```
/MyProject
│── CMakeLists.txt
│── main.cpp
│── my_library/
│   ├── CMakeLists.txt
│   ├── my_library.h
│   ├── my_library.cpp
│── cmake/
│   ├── custom_helpers.cmake
```
